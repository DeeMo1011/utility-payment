ตั้งค่าระบบ

วิธีตั้งค่า LINE Messaging API

  1. เข้า developers.line.biz/console/ → Login
  2. กด Create a new provider → ตั้งชื่อ
  3. กด Create a new channel → เลือก Messaging API
  4. กรอกข้อมูล Channel → กด Create
  5. ไปที่แถบ Messaging API → เลื่อนหา Channel access token → กด Issue → คัดลอกมาวาง
  6. หา User ID ของผู้รับ: เข้า LINE app → Profile → กด ··· → Share → จะเห็น ID ขึ้นต้นด้วย U
  7. ผู้เช่าต้อง Add Official Account เป็นเพื่อน ก่อนถึงจะรับข้อความได้
⚠️ Free tier ส่งได้ 200 ข้อความ/เดือน — เพียงพอสำหรับที่พักทั่วไป
ถ้าต้องการส่งมากกว่านี้ต้องสมัคร plan เพิ่ม
